# librandom

[librandom](https://github.com/cgwrench/librandom) implements a number of state
of the art, portable pseudo-random number generators in C for integer and
floating-point deviates drawn from various distributions.

[librandom](https://github.com/cgwrench/librandom) is released under the
permissive [BSD 2-Clause
License](http://opensource.org/licenses/bsd-license.php), see the file COPYING
for the license text.
